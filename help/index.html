Textacular Help

FAQ:

  • Q: How do I make a new file?

    A: Hover over the File button and select "New"

  • Q: What does "Save Locally" mean?

    A: The "Save Locally" feature allows you to save and load files from local storage

  • Q: All my files are gone!

    A: Textacular uses Local Storage to store your files online. If you clear this, ALL of your files will be gone!

  • Q: How do I view my text as HTML?

    A: Hover over the View button and select "HTML"

  • Q: How do I make my own theme?

    A: Open the Theme menu and fill out the boxes,
    once you are done you can use your theme any time by selecting "custom" from the themes menu

What is Textacular?

Textacular is an easy to use, online text editor. With Textacular you can open, edit and save text files.
You can also edit and view html files live with the HTML view option.

Why use Textacular? By using Textacular you can avoid the hassle of opening a program just to write a note.
You can also store all of your notes in one place and load them easily online.

How to make a theme:

Textacular themes are very simple to make. They consist of 2 elements, 1: The theme name and 2: the theme colors.
As of right now Textacular only supports HEX colors, but we may allow RGB in the future. Each theme contains 7 colors.

  • Background color

  • Editor color

  • Border color

  • Top menu color

  • Button hover color

  • Text color

  • Highlight color (Header)

Here is an example theme: ["My Theme", ["#111111", "#222222", "#000000", "#111111", "#333333", "#ff0000", "#ff0000"]].
If you enter an invalid theme, it will not show up on the themes menu. If your theme does not show up on the themes menu then 1. Your theme is invalid or 2. you cleared your local storage.
